Exploring DevTools

Intro

DevTools are a set of web developer tools built directly into the browser. It allows you to inspect and edit the HTML and CSS of a page, debug JavaScript, and view network requests.

It's a powerful tool that can help you understand how a website is built and how it works.

You can use them to find bugs and figure out solutions to them.

It's also a great way to learn about web development and to improve your own skills.
